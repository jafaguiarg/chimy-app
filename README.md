# This app is simply a time announcer

A Flutter app that announces the current time at configurable intervals using text-to-speech and system notifications.

## Features

- Configurable announcement intervals: 1, 5, 15, 30, or 60 minutes
- Text-to-speech announcements: "THE TIME IS [HH:MM]" in military format
- System notifications for each time announcement
- Works in background on Android and iOS
- Cross-platform support: macOS, Windows, iOS, and Android
